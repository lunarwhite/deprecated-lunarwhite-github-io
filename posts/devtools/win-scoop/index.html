scoop: command-line package manager for Windows

Posted on Nov 20, 2019

1 Do I really need？

众所周知，在Windows下进行开发多少有点反人类，环境配置和macos/linux相比，可以说是灾难。索性有人开源了一系列包管理工具，主要有scoop/chocolatey/winget。对比之下，选择了 scoop，体验良好，写一文向大家安利。

通过包管理工具，你可以直接在终端中快速自动地安装、配置、卸载和升级软件，而无需通过 GUI 页面一步步手动安装下载软件。这非常适合一次性安装非常多软件，大大地节约了软件安装所需要的时间和步骤，同时也在一定程度上确保了来源软件的安全性。

优点包括不限于：降低安装维护软件的成本，避免安装大量软件造成的路径污染，不必查找和安装软件的其他依赖项，避免捆绑和垃圾软件，卸载彻底。

2 Setup environment

set-executionpolicy remotesigned -scope currentuser

iex (new-object net.webclient).downloadstring('https://raw.githubusercontent.com/lukesampson/scoop/master/bin/install.ps1')
  • basic apps
scoop install 7zip
scoop install git
scoop install sudo
scoop install innounp
scoop install dark
  • checkup

5 Frequently used commands

# get
scoop search <app>
scoop info <app>

# bucket
scoop bucket list
scoop bucket known
scoop bucket add <bucket> <bucket_url>
scoop bucket rm <bucket>

# install & update
scoop install <app>
scoop update <app>

# uninstall
scoop uninstall <app> -p
scoop uninstall scoop

# check
scoop list
scoop status
scoop checkup

# clean
scoop cache rm *
scoop cleanup *
