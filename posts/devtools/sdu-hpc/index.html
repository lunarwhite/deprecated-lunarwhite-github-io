Available HPC server in SDU

Posted on Apr 21, 2021

1 配置

1.1 连接

  • ssh stu14@219.231.162.66
  • 默认：a123456

1.2 系统信息

  • 1个登录节点ln01
  • 1个管理节点mu01
  • 34个CPU节点cu01-cu34，队列名cu
  • 2个GPU节点gpu01、gpu02，队列名gpu
  • 2个mic节点mic01、mic02，队列名mic

1.3 环境变量

  • 编辑配置文件 vi ~/.bashrc
  • 写入环境变量
source /opt/intel/composer_xe_2015.1.133/bin/compilervars.sh intel64
source /opt/intel/impi/5.0.2.044/bin64/mpivars.sh
source /opt/intel/mkl/bin/intel64/mklvars_intel64.sh
  • 保存文件 :wq 后，使修改生效 source ~/.bashrc
  • 同一账号修改一次即可，但最后一步生效命令都需要做
  • 修改后，使用 which mpicc 命令检查是否完成环境变量配置
  • 可以找到编译器路径，表明配置成功
