Windows terminal

Posted on May 13, 2020

千呼万唤始出来，微软自家也有能称得上是terminal的terminal了

1 参考

  • 官方文档

2 注意

  • 确保添加bash到path
  • 获取 guid
  • 获取 icon(256*256)
    • Powershell：background1 background2
    • Windows：background1 background2
    • Git-Bash：background1 background2 icon1
    • Ubuntu：background1 icon1
    • Linux：background1 background2 background3

3 添加到右键菜单

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\Directory\Background\shell\wt]
@="Terminal Here"
"Icon"="C:\\Users\\mewin\\scoop\\apps\\windows-terminal\\current\\wt.exe"

[HKEY_CLASSES_ROOT\Directory\Background\shell\wt\command]
@="C:\\Users\\mewin\\scoop\\apps\\windows-terminal\\current\\wt.exe"
