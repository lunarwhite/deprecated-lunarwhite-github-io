4 函数与类

4.1 函数

  • 结构：
# 无参数
>>> def hello():
...     print("Hello world!")
...
>>> hello()
Hello world!

# 有参数
>>> def hello(object):
...     print("Hello "+object+"!")
...
>>> hello("cat")
Hello cat!
  • if __name__ == '__main__':
    • 这个是只有直接调用才能执行这句后面的代码。
    • 如果import本脚本，不执行这句后面的代码

4.2 类

  • 结构：
class 类名:
    def __init__(self,参数, …): #构造函数
    def  方法名1(self,参数,…): #方法1
    def  方法名2(self,参数,…): #方法2

# 定义
class Man:
    def __init__(self,name):
        self.name=name
        print("init success!")

    def hello(self):
        print("hello "+self.name+"!")

    def goodbye(self):
        print("goodbye "+self.name+"!")

# 调用
m=Man("David")
m.hello()
m.goodbye()
  • 用户可以定义类，创建数据类型、创建方法和属性

5.2 N维数组

import numpy as np

A=np.array([[1,2],[3,4]])
B=np.array([[3,0],[0,6]])

print(A*B) # 此处a*b是对应元素相乘
np.dot(A,B) # 这才是矩阵相乘

5.3 访问元素

import numpy as np

X=np.array([[51,55],[14,19],[0,4]])

# 访问某一行
print(X[0])

# 访问某一个
print(X[0][1])

# 循环访问
for row in X:
    print(row)

5.4 其他

import numpy as np

X=np.array([[51,55],[14,19],[0,4]])

X=X.flatten()
print(X)
# 输出一维
[51 55 14 19  0  4]

print(X[X>15])
# 输出 X 中 >15 的
[51 55 19]

print(X>15)
# 输出 bool 值
[True True False True False False]

6 工具包-Matplotlib

6.1 绘制

import matplotlib.pyplot as plt
import numpy as np

x=np.arange(0,6,0.1) # arange函数用于创建等差数组
y1=np.sin(x)
y2=np.cos(x)

plt.plot(x,y1,label="sin")
plt.plot(x,y2,linestyle="--",label="cos")

plt.xlabel("x")
plt.ylabel("y")
plt.title('sin & cos')

plt.legend()
plt.show()

6.2 读取显示

import matplotlib.pyplot as plt
from matplotlib.image import imread

img=imread('ball.jpg')
plt.imshow(img)
plt.show()
