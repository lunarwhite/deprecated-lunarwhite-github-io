Conda

Posted on Apr 12, 2020

1 基本环境

1.1 检查pip

python -m ensurepip
python -m pip install --upgrade pip

1.2 换镜像源

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2
conda config --set show_channel_urls yes

1.3 其他常用

# 创建环境
conda create --name <env_name> <package_names>

# 切换环境
activate <env_name>
deactivate <env_name>

# 显示已安
conda info --envs

# 复制，后删除
conda create --name <new_env_name> --clone <copied_env_name>
conda remove --name <env_name> --all

2 常用包库

2.1 工具包

pip3 install numpy
pip3 install opencv-python

pip3 install matplotlib
pip3 install scipy

pip3 install jupyter

pip3 install scikit-learn

2.2 keras框架

conda create -n <env_name> python=3.7 -i https://pypi.doubanio.com/simple

conda activate <env_name>

pip install tensorflow==2.2 -i https://pypi.doubanio.com/simple

pip3 install keras==2.3.1 -i https://pypi.douban.com/simple
import keras

conda deactivate <env_name>

2.3 keras可视化

pip3 install pydot
pip3 install pydot_ng
pip3 install graphviz
